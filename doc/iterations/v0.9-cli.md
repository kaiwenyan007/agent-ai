# v0.9 CLI 命令行客户端

> **本步目标**：对标 Python `main.py`，提供 full / simple 双模式 CLI。  
> **预计时间**：4–5 小时

---

## 前置条件

- [ ] 完成 [v0.6](./v0.6-tools.md)（Tool Calling）
- [ ] JDK 21、数据库（MySQL 或 H2）

---

## 架构

| 模块 | 职责 |
|------|------|
| `AgentChatRunner` | 从 ChatService 抽出的对话核心，Web + CLI 共用 |
| `agent-cli` | Picocli 命令、REPL、登录与配置向导 |
| `CliProperties` | simple 模式读取 `agent.cli.llm` / 环境变量 |

---

## 双模式

| 模式 | 启动 | 登录 | 配置 | 历史 |
|------|------|------|------|------|
| **full**（默认） | `mvn -pl agent-cli -am spring-boot:run` | 必须 | DB 用户配置；未配置走向导 | 持久化多会话 |
| **simple** | 加 `--simple` | 否 | 环境变量 / yml | 内存 |

---

## 命令一览

```powershell
# 默认：full 模式聊天（未登录会提示登录，未配置 API 会走配置向导）
mvn -pl agent-cli -am spring-boot:run "-Dspring-boot.run.profiles=h2"

mvn -pl agent-cli -am spring-boot:run "-Dspring-boot.run.profiles=h2" "-Dspring-boot.run.arguments=login"
mvn -pl agent-cli -am spring-boot:run "-Dspring-boot.run.profiles=h2" "-Dspring-boot.run.arguments=register"
mvn -pl agent-cli -am spring-boot:run "-Dspring-boot.run.profiles=h2" "-Dspring-boot.run.arguments=config"
mvn -pl agent-cli -am spring-boot:run "-Dspring-boot.run.profiles=h2" "-Dspring-boot.run.arguments=--simple"
```

---

## 验收标准

- [ ] full 模式：登录后未配置 API 会自动进入配置向导
- [ ] full 模式：配置完成后可流式聊天并调工具
- [ ] simple 模式：配置 `OPENAI_API_KEY` 后无需登录即可聊天
- [ ] `quit` / `/clear` / `/config` / `/logout` 命令可用
- [ ] 登录态写入 `~/.agent-ai/cli-session.json`
