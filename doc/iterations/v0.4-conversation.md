## 前置条件

- [x] 完成 [v0.3](./v0.3-api-config.md)
- [x] Node.js 20+ 已安装

---

## 后端步骤

### 步骤 1：实体与 Mapper

- `Conversation` → `conversations`
- `Message` → `messages`

### 步骤 2：接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/conversations` | 当前用户会话列表 |
| POST | `/api/conversations` | 新建会话 |
| DELETE | `/api/conversations/{id}` | 删除 |
| GET | `/api/conversations/{id}/messages` | 消息历史 |
| POST | `/api/conversations/{id}/messages` | 追加用户消息（v0.5 改由 chat 流写入） |

### 步骤 3：自动标题

首条用户消息用 Hutool：

```java
StrUtil.subPre(prompt, 30) + (prompt.length() > 30 ? "…" : "")
```

更新 `conversations.title`。

### 步骤 4：CORS

`WebMvcConfig` 允许 `http://localhost:5173`。

## 前端步骤

### 步骤 1：初始化 Vue 3

```powershell
cd frontend
npm create vite@latest . -- --template vue-ts
npm install axios pinia vue-router
```

### 步骤 2：页面路由

| 路由 | 页面 |
|------|------|
| `/login` | 登录/注册 |
| `/chat` | 聊天（占位） |
| `/settings` | API 配置 |

### 步骤 3：axios 封装

- `baseURL: http://localhost:8080`
- 请求头带 token
