## 实现步骤

### 步骤 1：Token 统计

表：`token_usage`

接口：

- `GET /api/stats/tokens/summary` — 总 token、费用
- `GET /api/stats/tokens/by-model` — 按模型
- `GET /api/stats/tokens/recent` — 最近记录

### 步骤 2：RAG 统计

表：`rag_queries`、`chroma_cache_events`

- 查询次数、命中率
- memory_hit / disk_hit / rebuild 次数

### 步骤 3：启动预热

对标 Python `startup_bootstrap`：

| 阶段 | 行为 |
|------|------|
| 登录前 | 不加载 Embedding |
| 登录后 | `@Async` 可选预热 Agent 模块 |
| 轻量检查 | `GET /api/system/bootstrap` 返回 phase |
| REBUILD / 首次 RAG | 才真正加载向量库 |

Vue 启动画面：轮询 bootstrap 状态，可「跳过等待」。

### 步骤 4：UI 打磨

- 暗色主题（对标黑客风，可简化）
- 侧边栏：CHAT / SETTINGS / KNOWLEDGE / STATS
- 记住登录（localStorage）

### 步骤 5：演示脚本

编写 `doc/demo-script.md`，列出验收用例。
