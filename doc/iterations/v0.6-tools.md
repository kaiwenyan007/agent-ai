## 实现步骤

### 步骤 1：@Tool 注解

```java
@Component
public class AgentTools {

    @Tool(description = "获取当前日期、时间和星期几")
    public String getCurrentTime() { ... }

    @Tool(description = "执行加减乘除，operation: add/subtract/multiply/divide")
    public String calculate(String operation, double a, double b) { ... }
}
```

### 步骤 2：注册到 ChatClient

Spring AI 1.x：

```java
ChatClient.builder(chatModel)
    .defaultTools(agentTools)
    .build();
```

Spring AI 2.0：改用 `ToolCallAdvisor`。

### 步骤 3：系统提示词

写入 `system` 消息，规则对标 Python `langchain_agent.py`：

- 问时间必须调 `getCurrentTime`
- 问天气必须调 `getTodayWeather`，未给城市要追问
- 问知识库文件列表必须调 `listKnowledgeFiles`
- 文档内容检索优先 `queryKnowledgeBase`

### 步骤 4：天气工具

Open-Meteo API，Hutool HTTP：

```
geo: https://geocoding-api.open-meteo.com/v1/search
forecast: https://api.open-meteo.com/v1/forecast
```
