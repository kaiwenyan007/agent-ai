## 步骤 1：选择数据库方式

### 方式 A — H2 免安装（最快）

```powershell
cd C:\Users\12782\dev\agent-ai
mvn -pl agent-server spring-boot:run "-Dspring-boot.run.profiles=h2"
```

### 方式 B — 本地 MySQL（推荐）

1. 安装 MySQL → [setup/02-mysql-install.md](../setup/02-mysql-install.md)
2. 执行脚本：

```powershell
mysql -u root -p < sql/schema.sql
```

3. 修改 `application-dev.yml` 中的用户名密码
4. 启动：

```powershell
mvn -pl agent-server spring-boot:run
```

## 验收标准

- [ ] `mvn -pl agent-server spring-boot:run` 启动无报错

- [ ] MySQL 方式：`SHOW TABLES` 可见 8 张表；或 H2 控制台可见表

---

## 常见问题

| 问题 | 处理 |
|------|------|
| 连不上 MySQL | 检查服务是否启动、密码、库名 `agent_ai` |
| 端口 8080 占用 | `server.port: 8081` 写入 `application-local.yml` |
| Maven 慢 | 配置国内 mirror（阿里云） |
