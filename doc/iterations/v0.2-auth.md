## 实现步骤

### 步骤 1：实体与 Mapper

创建：

| 文件 | 说明 |
|------|------|
| `entity/User.java` | 对应 `users`，`@TableName("users")` |
| `mapper/UserMapper.java` | `extends BaseMapper<User>` |

字段：`id, username, passwordHash, createdAt, updatedAt, deleted`

### 步骤 2：注册接口

`POST /api/auth/register`

请求体：

```json
{ "username": "demo", "password": "demo123456" }
```

逻辑：

1. `StrUtil.isBlank` 校验（Hutool）
2. 用户名 ≥2、密码 ≥6
3. `BCrypt.hashpw(password)` 存 `password_hash`
4. `userMapper.insert(user)`
5. 同事务插入 `user_api_configs`、`user_knowledge_configs` 默认行

### 步骤 3：登录接口

`POST /api/auth/login`

1. 按 `username` 查询
2. `BCrypt.checkpw(password, hash)`
3. 返回 token（首期可用 **JWT** 或 **Session ID**）

推荐首期：**Sa-Token** 或 **Spring Security + JWT**（二选一，文档实现时确定）。

### 步骤 4：登出

`POST /api/auth/logout`

### 步骤 5：鉴权拦截

未登录禁止访问 `/api/**`（放行 `/api/auth/**`、`/api/health`）。

## Hutool 用法

```java
import cn.hutool.crypto.digest.BCrypt;

String hash = BCrypt.hashpw(plainPassword);
boolean ok = BCrypt.checkpw(plainPassword, hash);
```
