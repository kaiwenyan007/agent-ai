# v0.3 每人 API 配置

> **本步目标**：登录用户可保存/读取自己的 LLM API 配置（Key、Base URL、Model）。  
> **预计时间**：2–3 小时

---

## 前置条件

- [x] 完成 [v0.2](./v0.2-auth.md)，登录态可用

---

## 实现步骤

### 步骤 1：实体与 Mapper

| 文件 | 表 |
|------|-----|
| `UserApiConfig.java` | `user_api_configs` |
| `UserApiConfigMapper.java` | |

### 步骤 2：接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/settings/llm` | 获取当前用户配置（Key 脱敏显示） |
| PUT | `/api/settings/llm` | 保存 apiKey、baseUrl、model |

注册时插入默认行：`base_url=https://api.deepseek.com`，`model=deepseek-chat`。

### 步骤 3：拉取模型列表（可选）

`GET /api/settings/models`

使用 Hutool `HttpRequest` 请求 `{baseUrl}/models`，Header 带 `Authorization: Bearer {apiKey}`。

失败时返回默认列表：`deepseek-chat` 等。

### 步骤 4：配置校验工具

```java
public boolean isApiConfigured(Long userId) {
    UserApiConfig cfg = ...;
    return StrUtil.isAllNotBlank(cfg.getApiKey(), cfg.getBaseUrl(), cfg.getModel());
}
```

v0.5 聊天前检查此条件。
